# NoteMark

A minimal Note taking app with out-of-the-box markdown support.

![Preview](https://github.com/LorenzoLoPresti/NoteMark/assets/0000.png)

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
