# NoteMark (Windows)

Un semplice editor di testo per creare e modificare note. Applicazione desktop per windows

![Preview](https://github.com/LorenzoLoPresti/images/blob/main/note-mark/image1.png)

## Stack Utilizzato

- Electron
- React
- Typescript
- Tailwind
- Lodash
- Jotai

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
